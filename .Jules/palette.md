## 2024-05-26 - Dynamic Button Accessibility & Focus States in Vanilla CSS
**Learning:** In vanilla HTML/CSS projects, standard interactive elements like buttons don't always inherit accessible keyboard navigation feedback. Additionally, dynamically created icon-only buttons via DOM manipulation easily miss crucial `aria-label` and `title` properties since they're constructed without declarative templates.
**Action:** When manually writing UI functionality or adding vanilla CSS, explicitly define state modifiers like `:focus-visible` (and ensure they use adequate high-contrast outlines/box-shadows) for keyboard accessibility. Moreover, whenever `document.createElement('button')` creates an icon-only element, ensure `setAttribute('aria-label')` and `title` are explicitly declared.
